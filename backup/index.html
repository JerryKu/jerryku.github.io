About Me

Hi, my name is Jerry and I am a recent graduate of University of California, Santa Cruz. Though my early projects were primarily mobile based, I am currently pursing a career in web development. I believe that you learn most by doing so I am constantly striving to learn new things.

My skills in web development are primarily self taught. Object oriented programming languages I am proficient in are Java and Javascript. Some web technologies and frameworks that I have learned to use thus far: HTML, CSS, NodeJS, Express, PostGresSQL, MongoDB, PolymerJS, Bootstrap, AJAX, JSON, jQuery.

I currently work part time as an IT Technician at Corotech, in my free time I work as a full stack developer for ClubHub, a company started by my brothers and I. This allows me to practice and enhance my software engineering skills while I pursue a full time web developer position.

ClubHub

Html JavaScript Polymer CSSBootstrap Cordova NodeExpress PostGresSQL

ClubHub is a web-based mobile application that allows users to easily order drinks from bars straight from their phones. No more waiting to get the bartender's attention when you're at the clubs. Just order from your phone and step up to the bar when its ready.


OnWatch

Java C Android Cloud Pebble Android Studio

OnWatch is a Pebble Smartwatch application that keeps your emergency contacts easily accessible at a click of a button. Users may indicate their distress level and the application will send a text message to their emergency contacts with their distress level and last known location.


For this project, I led a group of 4 other students in HACKUCSC 2016 and where we received third place out of 100 other groups.

Nosh

Java Android Android Studio Yelp API Uber API Python

Nosh is an Android application that allows users to randomly generate nearby restaurants based on preset parameters and have food delivered to them or call an uber to take them to the restaurant.


This is a project that was created during an Intro to Software Engineering Class. We utilized SCRUM and agile practices design and build our application.
